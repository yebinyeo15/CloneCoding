# Musign Clone Coding

## 프로젝트 소개

Musign 웹사이트를 참고하여 제작한 반응형 클론 코딩 프로젝트입니다.  
HTML, CSS, JavaScript를 사용해 메인 비주얼, 섹션별 콘텐츠, 메뉴 인터랙션, 스크롤 애니메이션을 구현했습니다.

## 사용 기술

- HTML5
- CSS3
- JavaScript

## 주요 기능

- 반응형 웹 레이아웃
- 햄버거 메뉴 열기/닫기 기능
- 스크롤 위치에 따른 섹션 애니메이션
- 스크롤 구간에 따른 배경색 변경
- SVG path 애니메이션
- 이미지와 텍스트를 활용한 포트폴리오형 메인 페이지 구성

## 실행 방법

1. 프로젝트 폴더를 다운로드하거나 클론합니다.
2. `musign/index.html` 파일을 브라우저에서 실행합니다.
3. 별도의 서버나 설치 과정 없이 바로 확인할 수 있습니다.

## 프로젝트 목적

이 프로젝트는 웹 퍼블리싱 실습과 포트폴리오 제작을 목적으로 진행되었습니다.  
정적인 HTML/CSS 구조를 기반으로 JavaScript를 활용한 메뉴 동작, 스크롤 이벤트, 애니메이션 처리 방식을 연습하는 데 중점을 두었습니다.
