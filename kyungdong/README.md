# 경동 사이트 클론코딩

HTML, CSS, JavaScript를 활용해 **경동 공식 웹사이트의 메인 페이지**를 클론코딩한 포트폴리오 프로젝트입니다.

---

## 📌 프로젝트 소개

본 프로젝트는 경동 공식 사이트의 레이아웃과 주요 UI를 분석하여 제작한 클론코딩 작업물입니다.

웹 페이지의 구조를 직접 설계하고, 메인 비주얼 영역의 슬라이드 인터랙션을 구현하며 실제 서비스와 유사한 사용자 경험을 구현하는 데 중점을 두었습니다.

- **원본 사이트**: http://www.kyungdong.co.kr/
- **작업 유형**: 웹사이트 클론코딩
- **제작 목적**: 취업 포트폴리오 및 퍼블리싱 역량 향상

- 메인 영역 슬라이드 구현
- 이전 / 다음 버튼을 이용한 슬라이드 이동
- 자연스러운 화면 전환 효과 적용

- Header
- Hero(Visual)
- Business Section
- Innovation Section
- Public Section
- About Section
- Footer

시맨틱 태그를 활용하여 구조적으로 마크업하였습니다.

### 🎨 CSS 스타일링
- 섹션별 레이아웃 구성
- 이미지 및 버튼 스타일링
- 컬러 시스템 적용
- 여백 및 정렬 최적화

### ⚙️ JavaScript 인터랙션
- 메인 슬라이드 기능 구현
- 이전 / 다음 버튼 이벤트 처리
- 사용자 인터랙션에 따른 화면 변경
